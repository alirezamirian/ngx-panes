Getting started

ngx-panes depends on Angular@5.x.x. It may also work with older versions of angular but it's not tested.

Installation

Install ngx-panes via npm: npm i --save ngx-panes Or via yarn: yarn add ngx-panes

Usage

1. Import NgxPanesModule: 2. Include a theme by either including ngx-pane-theme mixin in a globally applied style. Learn more about theming options. Or use a pre-built theme: 3. Use it inside your template: And it will result into: First pane Second pane Third pane Forth pane
